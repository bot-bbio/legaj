# Installing LeGaJ

**Windows:** Download and run `LeGaJ-Setup.exe` from the releases page — it installs automatically, no admin password needed.

**Mac:** Open Terminal (`Cmd + Space` → type "Terminal" → Enter) and run:
```
git clone https://github.com/bot-bbio/legaj.git
cd legaj
chmod +x install_mac.sh && ./install_mac.sh
```
The script handles everything automatically. Just follow any prompts it gives you.
